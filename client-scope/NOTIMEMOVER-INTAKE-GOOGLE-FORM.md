# NoTimeMover — client intake (Google Form blueprint)

**How to use:** Create a Google Form with these sections. Duplicate checkbox / multiple-choice options into Google Forms manually. Mark **Required** where indicated. Add a **File upload** question only if you're on Google Workspace (or use Drive link in a short-answer field).

**Form title:** NoTimeMover — Business & operations intake  
**Form description (intro):**  
> This form collects everything we need to complete your **website build-out** (copy, SEO, trust content) and **Google Business Profile** setup without endless email threads. Answers should reflect how you **actually operate today** (or how you will at launch). If something doesn’t apply, write **N/A**—don’t leave blanks without a note. Estimated time: **25–40 minutes**.

### Google Forms tips for Finn

- Turn on **Collect email addresses** if only Jermaine should submit; otherwise leave off for team submissions.
- Use **Sections** matching A–K with **progress bar**.
- For long lists (ZIPs), allow **“See link in response”** and add short answer for the URL.
- After submission: **auto-reply** confirming next steps (“We’ll follow up within X with any clarifying questions—ideally none if complete”).
- **Optional duplicate** lightweight form for “emergency updates” later (3 questions only) so they don’t resubmit the mega form for tiny edits.
